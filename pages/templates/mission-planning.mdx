## What's Included

This template guides you through:

1. **Mission Identity** — Title, subject, grade level, duration
2. **Driving Question** — Using the template structures from Module 3
3. **Context Scenario** — The real-world setting and problem
4. **Stakeholder Audience** — Who receives the final product
5. **Learning Goals** — Four-tier objectives aligned to standards
6. **Assessment Plan** — Rubric criteria and checkpoint schedule
7. **Student Roles** — Role descriptions and responsibilities
8. **Workbook Outline** — Page-by-page plan for the student workbook
9. **Resource List** — Materials, tools, and references needed
10. **Implementation Notes** — Scheduling, grouping, and logistics

## How to Use

1. Make a copy of the template document
2. Work through each section in order
3. Use the AI prompts provided in each course module to generate draft content
4. Review and revise all AI-generated content using your professional judgment
5. Share with a colleague for feedback before implementation

<Callout type="info" title="Tip">
Complete Modules 1-6 before attempting to fill in the entire template. Each module provides the knowledge and AI prompts needed for its corresponding template section.
</Callout>
