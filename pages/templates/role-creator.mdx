## What's Included

The worksheet walks you through designing 4-5 student roles for a PBL mission:

### Section 1: Context Analysis
- Mission context summary
- Key tasks the project requires
- Skills needed to complete the project

### Section 2: Role Identification
- Brainstorm real-world counterparts
- Select 4-5 roles that cover all project tasks
- Verify each role has a clear purpose

### Section 3: Role Descriptions
For each role:
- Title and real-world counterpart
- Purpose statement (one sentence)
- 3-5 key responsibilities
- Required deliverables
- Learning goals addressed (by tier)

### Section 4: Balance Check
- Workload comparison across roles
- Skill distribution across roles
- Learning goal coverage verification
- Overlap and gap analysis

## How to Use

1. Complete the Context Analysis section first to identify what the project needs
2. Brainstorm roles based on real-world professionals in your mission context
3. Write full descriptions for each role
4. Run the Balance Check to verify equitable distribution
5. Revise until all checks pass

<Callout type="warning" title="Common pitfall">
Avoid creating roles that are unequal in workload or learning value. Every role should develop meaningful skills and contribute substantively to the final product. The "Artist" who only decorates the poster is not an equitable role.
</Callout>
