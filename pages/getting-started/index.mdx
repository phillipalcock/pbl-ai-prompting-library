# Getting Started

This module introduces the AI-assisted PBL design process and helps you set up the tools you'll use throughout the course.

## What You'll Learn

- How AI fits into the PBL design workflow
- Setting up your AI tools and prompting environment
- The iterative design process for PBL with AI

## Prerequisites

- Basic familiarity with a web browser
- Access to an AI chatbot (ChatGPT, Claude, or similar)
- A curriculum standard or topic you want to teach through PBL
