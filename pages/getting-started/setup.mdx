# Setting Up AI Tools

Before you begin designing curriculum materials, you need a working AI environment.

## Recommended Tools

| Tool | Purpose | Free Tier |
|------|---------|-----------|
| ChatGPT | General content generation | Yes |
| Claude | Long-form writing, analysis | Yes |
| Midjourney/DALL-E | Visual assets | Limited |
| Canva | Layout and design | Yes |

## Setting Up Your Workspace

1. Create accounts for your chosen AI tools
2. Bookmark this course site for reference
3. Prepare a folder structure for your curriculum project files
4. Have your curriculum standards document accessible

## Prompting Basics

When working with AI for curriculum design, your prompts should include:

- **Role**: Tell the AI what expert perspective to use
- **Context**: Provide grade level, subject, and standards
- **Task**: Specify exactly what you want generated
- **Format**: Describe the output structure you need
- **Constraints**: Set boundaries (reading level, length, tone)
