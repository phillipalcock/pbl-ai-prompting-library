# Build PBL with AI

A practical course for educators who want to design rigorous, engaging project-based learning experiences using AI tools.

## What You'll Learn

This course walks you through every stage of PBL design — from writing driving questions to building student workbooks — with AI as your co-designer.

Each module includes templates, examples, and exercises you can use immediately in your planning.

## Course Modules

| Module | Topic | What You'll Build |
|--------|-------|-------------------|
| 1 | Getting Started | Your AI-assisted PBL workflow |
| 2 | Foundations | Core PBL framework understanding |
| 3 | Mission Design | Driving questions and real-world context |
| 4 | Learning Goals | Four-tier learning objectives |
| 5 | Assessment | Aligned rubrics and checkpoints |
| 6 | Student Roles | Authentic roles with responsibilities |
| 7 | Workbook Creation | 18-page student workbook |
| 8 | Visual Design | Professional layouts and graphics |
| 9 | Writing & Editing | Clear, student-facing language |
| 10 | Equity & Access | Inclusive design strategies |

## Who This Is For

- Teachers designing PBL units for the first time
- Experienced PBL practitioners looking to use AI tools effectively
- Curriculum designers building PBL resources at scale
- Instructional coaches supporting PBL adoption

## How to Use This Site

Work through the modules in order, or jump to the section you need. Each module is self-contained but builds on concepts from earlier sections.

Use the **Templates** section for ready-to-use planning documents. Check **Examples** for complete PBL missions you can reference or adapt.
