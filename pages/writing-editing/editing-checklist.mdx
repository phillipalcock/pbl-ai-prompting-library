# Editing Checklist

Use this checklist to review every page of your curriculum materials before finalizing.

## Content Check
- [ ] Every instruction is specific and actionable
- [ ] Language is appropriate for the target audience
- [ ] Technical terms are defined in context
- [ ] Reading level matches target audience
- [ ] Content aligns to stated learning goals

## Clarity Check
- [ ] Instructions follow a logical sequence
- [ ] Each page has a clear purpose
- [ ] Students can complete the page without additional verbal instructions
- [ ] Response spaces indicate expected length/format

## Accuracy Check
- [ ] All facts and data are verified
- [ ] Standards citations are correct
- [ ] Rubric criteria match learning goals
- [ ] Timeline is realistic for class schedule

## Accessibility Check
- [ ] Text has sufficient contrast against background
- [ ] Font size is at least 11pt for body text
- [ ] Images have descriptive alt text
- [ ] Color is not the only way information is conveyed
- [ ] Document structure uses proper headings

## AI-Specific Check
- [ ] No AI hallucinations or fabricated facts
- [ ] No placeholder text left in ("insert X here")
- [ ] Tone is consistent throughout (no personality shifts)
- [ ] No repetitive phrasing across pages
- [ ] Content reflects your specific context, not generic examples
